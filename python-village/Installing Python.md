<h1>Why Python?</h1>
<p>Rosalind problems can be solved using any programming language. Our language of choice is Python. Why? Because it's simple, powerful, and even funny.</p>

<p>After completing installation, launch IDLE (default Python development environment; it's usually installed with Python, however you may need to install it separately on Linux).</p>

<code> >>> </code>

<p>The three arrows are Python's way of saying that it is ready to serve your every need. You are in interactive mode, meaning that any command you type will run immediately. Try typing <code>1+1</code> and see what happens.</p>

<p>Of course, to become a Rosalind pro, you will need to write programs having more than one line.
So select <b>File → New Window</b> from the IDLE menu.
You can now type code as you would into a text editor. 
For example, type the following: </p>

<code>print('Hello, World!')</code>

Select <b>File → Save</b> to save your creation with an appropriate name (e.g., hello.py).
