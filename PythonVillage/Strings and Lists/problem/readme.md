

<h1>Problem</h1>
<p><strong>Given</strong>: A string <em>s</em> of length at most 200 letters and four integers <em>a, b, c</em> and <em>d</em>.</p>

<p><strong>Return</strong>: The slice of this string from indices <em>a</em> through <em>b</em> and <em>c</em> through <em>d</em> (with space in between), <em>inclusively</em>.  In other words, we should include elements <strong><em>s[b]</em></strong> and <strong><em>s[d]</em></strong> in our slice.</p>


<h2>Sample Dataset</h2>
<strong><pre><code>HumptyDumptysatonawallHumptyDumptyhadagreatfallAlltheKingshorsesandalltheKingsmenCouldntputHumptyDumptyinhisplaceagain.
22 27 97 102</code></pre></strong>

<h2>Sample Output</h2>
<strong><pre><code>Humpty Dumpty
</code></pre></strong>
