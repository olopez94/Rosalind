<h1>
  Variables and Some Arithmeticclick 
</h1>

<p>
One of the most important features of any programming language is its ability to manipulate variables. A <strong>variable</strong>
is just a name that refers to a value; you can think of a variable as a box that stores a piece of data.
</p>

<p>
In Python, the basic data types are strings and numbers. 

There are two types of numbers:
<strong>integers</strong> (both positive and negative) and <strong>floats</strong> (fractional numbers with a decimal point).
You can assign numbers to variables very easily.
</p>

<pre><code>a = 324
b = 24
c = a - b
print("a - b is ", c)
</pre></code>

<p>
In the above code, a, b, and c are all integers, and 'a - b is' is a string.  The result of this program is to print:
</p

<pre><code>
a - b is 300

</pre></code>


<p>
You can now use all common arithmetic operations involving numbers:
</p>
<ul>
  <li> Addition: <strong> 2 + 3 </strong> </li>
  <li>Subtraction: <strong>5 - 2</strong></li>
  <li>Multiplication: <strong> 3 * 4</strong></li>
  <li>Division: <strong>10 / 2</strong></li>
  <li>Division Remainder: <strong>18 % 5</strong></li>
  <li>Exponentiation: <strong>2 ** 3</strong></li>

</ul>
